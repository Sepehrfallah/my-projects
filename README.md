# 📱 Notes App (Flutter)

## 📌 Overview
This is a simple notes application built using Flutter.

## 🚀 Features
- Add new notes
- Search notes
- Mark notes as favorite ⭐

## 🛠 Technologies Used
- Flutter
- Dart

## 📱 Pages
- Home Page
- Add Note Page

## 🧠 How it works
Users can add notes, search through them, and mark important notes as favorite.

This project was created for the COM206 course.
